# Changelog

Все заметные изменения документируются в этом файле.

## [1.1.1] - 2026-04-02

### Fixed
- CI: release notes передаются через файл (fix кириллицы в CHANGELOG)

## [1.1.0] - 2026-04-02

### Added
- Linked user — dropdown с пользователями платформы при создании пациента (human)
- Emoji picker для аватара (preset'ы human/animal + ручной ввод)
- Подтверждение удаления (confirm dialog) для всех сущностей
- Валидация обязательных полей с визуальной индикацией ошибок
- Контекстные подсказки: "Выберите пациента", "Выберите назначение", "Нет медикаментов для этого типа"

### Changed
- Улучшены labels форм: "Имя пациента", "Название медикамента", placeholder'ы с примерами
- Дни недели в списке расписаний теперь переводятся (Пн, Вт... вместо mon, tue...)

### Fixed
- Генерация доз: fix для MaxOpenConns=1 (rows закрываются перед insert)
- Начальная генерация доз отложена в background ticker (не блокирует запуск)

## [1.0.0] - 2026-04-02

### Added
- Пациенты (human/animal, привязка к пользователю системы)
- Глобальный справочник медикаментов (target type, дозировка, форма)
- Назначения (пациент + медикамент + дозировка + период)
- Расписания (время + дни недели)
- Автогенерация доз из расписаний (ticker каждые 5 мин)
- Overdue detection (>30 мин)
- Виджет Today: список доз, переключение пациента (per-board), progress bar, inline confirm
- Страница настроек: CRUD для всех сущностей
- WebSocket события (dose.given, dose.skipped, dose.overdue, dose.reminder)
- i18n (en, ru)
